## Install

```shell
npm install --save vovanr-project-logos
```

```js
var foo = require('vovanr-project-logos');

foo()
//=> Foo
```

## Development

```shell
git clone git@github.com:VovanR/vovanr-project-logos.git
cd vovanr-project-logos
npm install
npm start
```

Open https://localhost:8080
